Assignment 10 - Tables

For this assignment, you will create a basic HTML Page called Assignment10.html that contains a <table>.

Requirements

Your webpage should have the following content:

A Heading: "<h1>Monthly bills</h1>"

A <table> with the following:

  • 4 columns with these 4 table headings: Bill, April, May, June
  • 1st Row: Phone, $120, $140, $140
  • 2nd Row: Gas, $90, $100, $150
  • 3rd Row: Food, $280, $280, $300

Remember your table consists of : table header tags (<th>), table row tags (<tr>), and table data tags (<td>)
