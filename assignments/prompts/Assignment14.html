Assignment 14 - Forms

Basic Requirements

Create a new basic HTML page called Assignment14.html in your /assignments folder.  

Your page should contain a <form> tag, with the following children (as a bare minimum):

  • 3 <input> tags of type "text", each with a different name attribute
    • e.g. <input type="text" name="myInput">
  • 3 <input> tags of type "radio", with identical name attributes and different value attributes
    • e.g. <input type="radio" name="myRadioInput" value="option1">
  • 1 <input> tag of type "submit", with a value attribute to specify different text other than "Submit"
    • e.g. <input type="submit" value="Click to submit this form!">

Challenge Tasks

  • Use a <fieldset> with a <legend> in your form
  • Use other input types like 'checkbox', 'button', or 'reset'
  • Use other form elements like <textarea>, or <select> and <option>
  • Change the padding and border of your 'text' type <input> elements
  • Change the border-radius and background of the submit button (consider trying an image or gradient)
  • In general, make the form look more "fancy"!
