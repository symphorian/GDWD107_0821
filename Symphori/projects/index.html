This page contains several of my favorite projects from the last couple years. Some of them were for school; some of them were simply for my own amusement or to satisfy my curiosity.

  blink - Status: Ongoing (last updated June 2011)
A 2D platforming game engine for Flash; includes physics, gesture recognition, collision detection between rectangular objects with any rotation, and a tile-based level editor with different layers of depth and the ability to import images. (solo project, 6700 lines of ActionScript 3.0) - Play game demo - View demo info - Try level editor		  
  The Dark Lord's Apprentice - Status: Ongoing (last updated June 2011)
A variation on the tower-defense genre; the player can defend their character by summoning minions or creating trails of fire on the field; as the player levels up, they grow more powerful and can summon new types of minions with different abilities and attack patterns. (4-person group project, contributed 3200 lines of C# code myself, using the Microsoft XNA Game Studio library) - View game info - View installation info		  
  Stack Comics - Status: Ongoing (last updated June 2010)
An experiment in a new method of story-telling, which I call Stack Comics; the story for this Stack Comic is a reinvention of the old fable of Little Red Riding Hood. (solo project, 1000 lines of ActionScript 3.0 for the Stack Comic editor and player; 70 hand-drawn panels for demo, edited using Adobe Illustrator) - View info - View story		  
  Breeze - Status: Completed
An artistic game where up to two players can each control a fan which can be used to keep various floating objects aloft, such as leaves, snowflakes, flower petals, and cinders; blowing puffs of air at these objects will cause a sound to play and may cause the environment to change. The game's emphasis is on playing, rather than winning. (solo project, 500 lines of ActionScript 3.0 code) - View game info - Play game		  
  Hymmnos Jisho - Status: Ongoing (last updated March 2011)
An interactive dictionary to supplement the learning of Hymmnos, a fictitious language created by Akira Tsuchiya for the video game series Ar tonelico. Supports search by spelling and definition; can also filter search results by usage or dialect. (solo project, 500 lines of ActionScript 3.0 code) - Use Hymmnos Jisho - View info		  
  Weather - Status: Ongoing (last updated April 2011)
A simple weather app I made for myself utilizing Google's unofficial weather API; the application tells the current weather conditions for Northwestern University as well as the predicted weather for today and the next three days. (solo project, 250 lines of ActionScript 3.0 code) - View weather forecast		  
  BlobBugs - Status: Ongoing (last updated June 2011)
A class of customizable particles with attributes such as length, speed, color, gradient fill, and randomness. You can set their behavior so that they wander around a given rectangular area or target a single point. (solo project, 400 lines of ActionScript 3.0 code) - View demo		  
  Growth - Status: Ongoing (last updated December 2010)
A project intended to animate plant growth at a reasonable level of believability; utilizes a class with attributes the user can modify to influence the plant growth, such as whether the plant components are stiff or not, the max angle the components can grow at, the probability that a component will diverge, and whether there is a bound to the plants growth. (solo project, 550 lines of ActionScript 3.0 code) - View demo		  
  Light Lavalamp - Status: Ongoing (last updated January 2011)
A really simple project using a class originally intended to mimic the effect of light streaming through the clouds or from the surface to the ocean floor; the demo shows an incomplete version of the class with color instead. (solo project, 150 lines of ActionScript 3.0 code) - View demo
