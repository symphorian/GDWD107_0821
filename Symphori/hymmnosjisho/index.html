Hymmnos Jisho
Status: Ongoing (last updated March 2011)
 

Description:
One of my favorite video game series, Ar tonelico, has this really cool fictitious language known as Hymmnos, created by Akira Tsuchiya. The language simultaneously functions as a means for a singer to express their emotions and as a programming interface to interact with the Towers found in each game.

Hymmnos is a fully functional language with different dialects, complex grammar rules, and a fairly large vocabulary. This jisho (dictionary) is meant to help Ar tonelico fans learn and translate the Hymmnos language. It's not perfect yet, so if you have any suggestions for additional features, please let me know!
