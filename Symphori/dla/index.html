The Dark Lord's Apprentice
Status: Ongoing (last updated June 2011)

Description:
You are the cute little apprentice of the Dark Lord, who is off trying to conquer the world. While he is away, you have to protect his castle from the nasty heroes trying to take advantage of the situation. To do so, you can roast the heroes with fire, or summon minions to do the dirty work for you. The more heroes you take care of, the eviler you become, with new special powers to boot.

Installation Info:
1. Visit Microsoft's download page for XNA Game Studio 4.0 and follow the instructions there to install it. (Actually, as long as you have the the latest version of the .NET Framework, I don't think you need to install Visual Studio, but don't quote me on that)
2. Download TheDarkLordsApprentice.ccgame and save it to your desktop. The XNA Game Studio has a special way of packaging game projects so that other XNA Game Studio users can also play them, and that special way is to package them as .ccgame files.
3. Once you've downloaded the .ccgame file, double-click it, and select 'Unpack'. This will install the game to XNA's default folder on your computer; then, simply run the DarkLordsApprenticeV2.exe, and have fun!
